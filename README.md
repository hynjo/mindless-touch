# mindless-touch

소리만 듣고 화면을 탭해 숨겨진 도형을 찾는 모바일 웹 프로토타입입니다.

## 실행

Node.js 26.5.0을 사용합니다. 처음 한 번 의존성을 설치한 뒤 Vite 개발 서버를 실행하세요.

```bash
npm install
npm run dev
```

터미널에 표시된 로컬 주소를 열고 Chrome DevTools의 Device Mode를 켭니다.

## 플레이 방법

1. 빈 화면을 처음 탭하면 오디오가 활성화되고 문제 소리가 납니다.
2. 화면을 탭해 숨겨진 도형을 찾습니다. 오답이면 낮은 소리가 납니다.
3. 정답이면 성공 소리와 함께 도형, 오답 탭, 정답 탭이 표시됩니다.
4. 화면을 다시 탭하면 다음 문제가 시작됩니다.

드래그와 멀티터치는 판정하지 않습니다.

## 디버그

- `?debug=1`: 문제를 푸는 동안에도 정답 도형 표시
- `?seed=example`: 새로고침해도 같은 문제 순서 재현
- 함께 사용: `?debug=1&seed=example`

## 테스트

```bash
npm test
```

## GitHub Pages

프로덕션 파일을 생성합니다.

```bash
npm run build
```

결과물은 `dist/`에 생성됩니다. Vite의 `base`를 상대 경로로 설정했으므로 저장소 루트와 프로젝트 Pages 하위 경로 모두에서 에셋을 정상적으로 불러옵니다. GitHub Pages에는 `dist/` 결과물을 배포하면 됩니다.
